Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Common command:

  1. Create a new repository from the command line and associate it to the remote branch

#Associate and unlink remote repositories:
Git remote add origin https://xxxxxxx/xxxxx/helloworld.git
Git remote remove origin

View and delete remote branches:

Git branch -r
Git branch -r -d origin/branch-name
Git push origin :branch-name

#force push
Git push -u origin master -f
Where -u means –set-upstream
-f means –force

Pull a private github project

Two way:
a. Share the git address of the private repository;
Share the github account and password of the private repository owner.
Git clone + private repository git address: Clone with HTTPS to pull the project repo.
After waiting, you will be prompted to enter your account name and password.
b. Configure multiple RSA public keys to github
Each developer generates its own RSA key pair on its own computer and configures the public key into the github account that created the private repository, after which it can freely clone and submit the private repository just like the github account owner.
