git

2019-05-13

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

常用命令：

  1. 从命令行创建一个新的仓库，关联到该远程分支

#关联和取消关联远程仓库：
git remote add origin https://xxxxxxx/xxxxx/helloworld.git
git remote remove origin

查看和删除远程分支：

git branch -ｒ
git branch -r -d origin/branch-name
git push origin :branch-name

强制推送

git push -u origin master -f
其中的-u 表示 –set-upstream
-f 表示 –force

拉取私有github 项目

two way:
a.共享private repository 的git地址；
共享该private repository所有者的github账户及密码。
git clone + private repository的git地址:Clone with HTTPS的方式拉取项目repo。
稍等后会提示输入账户名和密码，输入即可
b.配置多个RSA公钥到github
每个developer在自己的电脑上生成自己的RSA秘钥对，并把公钥配置到创建该private repository的github账户中，此后即可像github账户拥有者一样自由克隆并提交该private repository。
